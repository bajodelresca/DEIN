Formulario

Aquí podemos registrar un nuevo instrumento

Requisitos

Todos los campos son requeridos para poder enviar el formulario, el único innecesario es la imagen.

Campo fecha

Para ingresar la fecha, solo debemos pulsar sobre el icono del calendario para que se nos abra una ventana de calendario

Spinner

Podemos añadir una nueva opción al spinner, pulsando en el icono "+" que encontramos a la derecha

Botones

En la parte inferior encontraremos estos tres botones, el primero para limpiar el formulario, el segundo para guardar, y un trcer boton para eliminar
